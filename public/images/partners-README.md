# Partner Placeholder Images

Le immagini dei partner devono essere inserite qui con i seguenti nomi:

- `partner1.png` - Logo Partner 1
- `partner2.png` - Logo Partner 2  
- `partner3.png` - Logo Partner 3
- `partner4.png` - Logo Partner 4

Specifiche:
- Formato: PNG con sfondo trasparente
- Dimensioni consigliate: 200x80px
- Logo su sfondo trasparente

Per ora vengono mostrati placeholder bianchi. Una volta aggiunte le immagini reali dei partner, sostituire i nomi file nel componente AboutPage.jsx.
