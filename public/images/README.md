# Immagini del Team

Inserire qui le foto del team con i seguenti nomi:

- `sharon-abate.jpg` - Foto di Sharon Abate (Amministratore delegato)
- `sara-incipini.jpg` - Foto di Sara Incipini (Coordinatrice Malatesta e Torre Maura)  
- `chiara-turco.jpg` - Foto di Chiara Turco (Coordinatrice Ponte di Nona)

Le immagini dovrebbero essere:
- Formato: JPG o PNG
- Dimensioni consigliate: 300x300px (quadrate)
- Formato ritratto professionale

Per ora vengono mostrate le emoji come placeholder. Una volta aggiunte le foto reali, sostituire nel file AboutPage.jsx le sezioni member-avatar con:

```jsx
<img src="/images/nome-file.jpg" alt="Nome Persona" className="member-image" />
```

Il CSS è già configurato per gestire correttamente le immagini circolari.
